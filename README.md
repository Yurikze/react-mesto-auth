# Проект: Место

### Описание

Социальная сеть для путешественников. Добавляйте фотографии посещенных мест.

**Тахнологии**

- React.js
- HTML
- CSS
- JS ES6
- Flexbox
- Grid Layout
- Responsive

**Доступные команты**

### `npm start`

Запускает приложение в режиме разработки.

### `npm run build`

Собирает приложение на продакшен.
